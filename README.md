# YCake_app

## 1. Установка используемых библиотек:
```
pip3 install python-telegram-bot 
pip3 install sqlite3 
```

## 2. Создать бота через [@botfather](https://t.me/botfather) и получить токен, полученный токен вставить в файл token.txt

## 3. Для запуска бота:

```
python3 main.py
```
